# Panduan Update ke Konten Organizer Lite

1. Ekstrak ZIP `konten-organizer-lite-github.zip`.
2. Masuk repository GitHub `konten-organizer`.
3. Klik `Add file` > `Upload files`.
4. Upload isi folder hasil ekstrak, terutama folder `.github` dan `dist`.
5. Klik `Commit changes`.
6. Buka tab `Actions` dan tunggu proses `Deploy Konten Organizer Lite` sampai hijau.
7. Buka kembali link GitHub Pages:
   `https://msukri89.github.io/konten-organizer/`
8. Jika masih tampak versi lama, tekan `Ctrl + F5` atau buka Incognito.

Catatan: data versi lama mungkin dimigrasi sebagian ke versi Lite. Jika ingin benar-benar bersih, buka menu Kalender > Reset.
